Physical Content of the delivery

Morpho Customer Support Package v2.8.zip
This package contains the following directories:

  • Code_Sample - CSP code source
  • Demo_Application - Apk built from CSP code source. (Iris and MultiSim demo must be signed to be working)
  • Customization package - Preinstalling applications on tablet, and signing them.
  • Documentation - ACS SmartCard, MorphoTablet CSP, MorphoSmart and MobIris programmer guides
  • Sdks - ACS SmartCard, MorphoSmart and MobIris libraries
