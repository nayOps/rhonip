MorphoTablet CSP Programming Guide

Samples

MorphoTablet Customer Support Package provides a set of source code samples related to different peripheral components of the device.
Code sample section of each feature describes how to create an application (Fingerprint sensor, SmartCard and so on).
Samples are available for the following features:

  • Bluetooth
  • Camera
  • Camera LED
  • External mass storage
  • Fingerprint sensor
  • Iris sensor (MorphoTablet™ 2i)
  • Microphone
  • Multi Sim
  • NFC
  • Open Mobile API
  • Peripheral power management
  • Physical Protection
  • Sam card
  • Smart Card
  • Stylus
  • Tablet Name
  • Tri-color LED
  • WLan/4G
