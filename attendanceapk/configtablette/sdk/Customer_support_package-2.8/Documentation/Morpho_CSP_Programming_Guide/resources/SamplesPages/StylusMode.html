The device has a stylus mode to make the touch panel more accurate. To enable stylus mode, developers have to add the two following permissions to the manifest: 

<uses-permission android:name="android.permission.READ_SETTINGS">
<uses-permission android:name="android.permission.WRITE_SETTINGS">

I) Initialization

Description – Instantiate the following string that is used to know the status or to switch ON/OFF the Stylus mode

public static final String ACCESSIBILITY_TOUCH_PANEL_ENABLED = "accessibility_touch_panel_enabled";

II) Get the Stylus mode status

Description – The following code gets the Stylus mode status. 

try {

    int valueStylus = Settings.Secure.getInt(getContentResolver(), ACCESSIBILITY_TOUCH_PANEL_ENABLED);
    switch_mode.setChecked(valueStylus==1);
} catch (Settings.SettingNotFoundException e) {
    e.printStackTrace();
}

III) Enable/disable the Stylus mode

Description – The following code enables/disables the Stylus mode. This method is available in software TA0 and higher.

Parameters:
ContentResolver cr - ContentResolver.
String name - string to select a Stylus mode ("accessibility_touch_panel_enabled").
int value - one to power on the device or zero to power off.

//Object of class Settings.System has method putInt(ContentResolver cr, String name, int value)
//this is the main function that we are using in our sample to switch on or off devices
Settings.System.putInt(getContentResolver(),ACCESSIBILITY_TOUCH_PANEL_ENABLED, value);

IV) Enable/disable the Stylus mode manually

Description – The following code starts android Settings to let the user switch ON/OFF the Stylus mode.

Intent intent = new Intent(Settings.ACTION_ACCESSIBILITY_SETTINGS);
startActivity(intent);
