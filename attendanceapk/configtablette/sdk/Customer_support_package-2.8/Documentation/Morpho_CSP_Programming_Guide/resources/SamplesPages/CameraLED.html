Functions

The device has two cameras, one 2.0 MP front camera, and one 13.0 MP rear camera. In order to use the camera, please call the following set of available functions. Sample uses Android library and custom methods. In order to use the "Turn on/off" feature of any camera LED, developers need to add the following permission to the manifest:

<uses-permission android:name="android.permission.CAMERA">

I) Get the method to set the value of flash LEDs

Description – The following API "Method" is used to invoke the method to set the value of flash LEDs.

private Method MTAB_SetFlashValues= null;
private void createMorphoTabletReflectionMethods() {
    if (camera == null) {
        camera = Camera.open();
    }

    Class<? extends Camera.Parameters> c = camera.getParameters()
            .getClass();
    Class[] cArg1 = { int.class, int.class };

    try {
        MTAB_SetFlashValues = c.getMethod("setFlashValue", cArg1);

    } catch (NoSuchMethodException | IllegalArgumentException e) {

        e.printStackTrace();
    }
}

II) Set Led Brightness

Description – Set the brightness of the flash LEDs. Parameters:

String device - the target led ("led1" or "led2").
int value - the target brightness value [0 ≤ value ≤ 225].
public void setLedPower(String device, int value) {
    parameters = camera.getParameters();

    if (device.equals(led1)) {
        valueFlash2 = value;
    } else {
        valueFlash1 = value;
    }
    try {
        MTAB_SetFlashValues.invoke(parameters,valueFlash1, valueFlash2);
    } catch (IllegalAccessException | InvocationTargetException e) {
        e.printStackTrace();
    }
    try {
        MTAB_SetFlashTorch.invoke(parameters,true);
    } catch (IllegalAccessException | InvocationTargetException e) {
        e.printStackTrace();
    }
    camera.setParameters(parameters);
}
