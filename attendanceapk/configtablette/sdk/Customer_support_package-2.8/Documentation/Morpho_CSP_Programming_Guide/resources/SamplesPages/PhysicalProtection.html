This sample demonstrates how to detect physical intrusion.

1) Receive intrusion event

Description – To enable physical intrusion detection, you have to add the following configuration to the manifest:

<receiver
    android:name=".TamperReceiver"
    android:enabled="true"
    android:exported="true">
    <intent-filter>
        <action android:name="com.morpho.morphotabletv2.tamper.changed" />
    </intent-filter>
</receiver>

Description – The intent contains the date of the intrusion (or null if tablet was off).

Parameters :
String eventTime 

public void onReceive(Context context, Intent intent) {

    final String intentAction = intent.getAction();
    Log.d(TAG, "Intent: a = " + intentAction + " " + intent.toString());

    //Get the time when the intrusion occurred
    String eventTime = intent.getStringExtra("eventTime");
    String msg;

    if(eventTime == null)
        msg = "INTRUSION HAS OCCURRED PREVIOUSLY";
    else
        msg = "INTRUSION JUST OCCURRED at date " + eventTime + " !";
        
        ...
        
}
