Functions

The MorphoTablet device has its own device name that you are able to get it read or modified. In order to use those features, call the available functions.

Get the Morpho Device Name

Description – Get the current Tablet Name. 

public void getName(View v){
        Toast.makeText(this,Settings.System.getString(
                getContentResolver(),MORPHO_DEVICE_NAME),Toast.LENGTH_LONG).show();
    }
}

Set the Morpho Device Name

Description – Set the Tablet Name. 

public void setName(View v){
        if (!newName.getText().toString().isEmpty()){
            Settings.System.putString(
                    getContentResolver(),MORPHO_DEVICE_NAME,newName.getText().toString());
        } else {
            Toast.makeText(this,"EditText is empty",Toast.LENGTH_LONG).show();
        }
    }
}
