Understanding Bluetooth Sample Application

Click to View Source

Bluetooth sample application is the part of Customer support package which demonstrates how to enable/disable bluetooth and search for bluetooth devices. This sample uses android sdk-21 library.

Functions

Developers must add the following two permissions to the manifest file: 

<uses-permission android:name="android.permission.BLUETOOTH">
<uses-permission android:name="android.permission.BLUETOOTH_ADMIN">

I) Initialization

BluetoothAdapter is an open android-based class which is used to perform all the operations related to bluetooth.

Description – This method is used to create an object of type BluetoothAdapter.

mBluetoothAdapter = BluetoothAdapter.getDefaultAdapter();

II) Enable/Disable bluetooth

Description – This sample code shows you how to switch between those two states.

if (mBluetoothAdapter.isEnabled()) {
    mBluetoothAdapter.disable();
} else {
    Intent intent = new Intent(BluetoothAdapter.ACTION_REQUEST_ENABLE);

    startActivityForResult(intent, 1000);
}

III) Scan bluetooth

When bluetooth is enabled, the area can be scanned with the following code:

Description – Find bluetooth devices in range.

mBluetoothAdapter.startDiscovery();

Description – Stop the search of devices in range.

mBluetoothAdapter.cancelDiscovery();

IV) Pair/Unpair bluetooth device

Description – the code below will pair/unpair a device. 

/** To pair device programmatically */
private void pairDevice(BluetoothDevice device) {
  try {
    Method method = device.getClass().getMethod("createBond", (Class[]) null);
    method.invoke(device, (Object[]) null);
  } catch (Exception e) {
    Log.e(TAG, "pairDevice", e);
  }
}
/** To unpair device programmatically */
private void unpairDevice(BluetoothDevice device) {
  try {
    Method method = device.getClass().getMethod("removeBond", (Class[]) null);
    method.invoke(device, (Object[]) null);
  } catch (Exception e) {
    Log.e(TAG, "unpairDevice", e);
  }
}
