Understanding Microphone Sample Application

Click to View Source

Microphone sample demonstrates a set of basic use cases: audio recording and playback. Sample are based on the standard Android library. Developers first have to add the following permission to the manifest file: 

<uses-permission android:name="android.permission.RECORD_AUDIO">

I) Initialization

Description – This method is used to create an object of MediaPlayer class.

/** To initialize player */
public void initPlayer() {
    mPlayer = new MediaPlayer();
    mPlayer.setOnCompletionListener(new OnCompletionListener() {

        @Override
        public void onCompletion(MediaPlayer mp) {
            mPlayButton.setText("Start playing");
            mStartPlaying = !mStartPlaying;
            isPlaying = false;
            updateDisplay("Stopped", 0.0);
            onPlay(false);
        }
    });
}

II) Start Playing

Description – Start playing from an audio source 

// setting selected location to play
mPlayer.setDataSource(selected_location);
mPlayer.prepare();
mPlayer.start();

III) Stop Playing

Description – Stop playing, releasing all resources attached to MediaPlayer object.

if(mPlayer!=null){
    mPlayer.reset();
    mPlayer.release();
    mPlayer = null;
}

IV) Start Recording

Description – Start recording the voice.

mSensor.start(mRecordedFileLocation);
setActivityLed(true);
mHandler.postDelayed(mPollTask, POLL_INTERVAL);
imgAudio.setVisibility(View.VISIBLE);

V) Stop Recording

Description – Stop recording the voice.

mSensor.stop();  //Stopping the recording

//Remove handler call back
mHandler.removeCallbacks(mPollTask);
mSensor.stop(); //Stopping the recording
