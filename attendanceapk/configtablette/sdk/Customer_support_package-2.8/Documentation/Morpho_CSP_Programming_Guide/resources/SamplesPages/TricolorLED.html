Understanding Tri-Color LED Sample Application

Functions

The device has a Tri-Color LED (notification LED). In order to set this LED color, call the available function.

Note: Charging Battery Led color can also be bypassed (refer to 'override' parameter).

Switch the Tri-Color LED On

Description – Set the LED color. 

void setColor(int c, boolean override)
    {
        if (override)   c = c & 0xFEFFFFFF;
        //Toast.makeText(getApplicationContext(), "Set Color " + c, Toast.LENGTH_SHORT).show();
        notification.ledARGB = c;
        notificationManager.cancel(1);
        notificationManager.notify(1, notification);
    }
}
