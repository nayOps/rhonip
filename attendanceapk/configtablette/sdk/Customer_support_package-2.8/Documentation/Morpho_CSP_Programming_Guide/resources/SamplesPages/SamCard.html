Understanding Sam card Sample Application

Click to View Source

Sam card sample is basically structured in two major parts:

a) OpenMobileAPI.jar

This is a primary jar which contains all the methods related to sam card operations.

b) Sample application

Sample application will provide a basic usage of Sam card by explaining the following features:

  • Import OpenMobileAPI.jar into sample application inside libsfolder.
  • Open, Close Reader and other operations related to Sam card
  • Transmit APDU commands and get response.

    Functions

    I) Initialization

    Description – Initialize the Sam card environment

    SEServiceCallback callback = new SEServiceCallback();
try {
    new SEService(getApplicationContext(), callback);
} catch (SecurityException e) {
    Log.d(TAG,"Binding not allowed, uses-permission org.simalliance.openmobileapi.SMARTCARD?");
} catch (Exception e) {
    Log.d(TAG,"Exception: " + e.getMessage());
}
    /** Used to connect to OMA service. */
public class SEServiceCallback implements SEService.CallBack{
    @Override
    public void serviceConnected(SEService seService) {
        Reader[] readers = seService.getReaders();
        for (Reader reader : readers) {
            if (reader.getName().equals("SAM")) {
                /** Initialization of SAM Reader **/
                mSamReader = reader;
            }
        }
        updateConsole(mSamReader.getName()+" reader ready to be used");
    }
}

    Description – Power On Sam card

    public int performPowerTaskOnSamCard() {
    try {
        /** Modify "POWER_ON_ISO" to choose a different mode **/
        ATR = mSamReader.PowerOnByMode(POWER_ON_ISO);
        if (ATR == null) {
            status = CloseReader();
            Message msg = mHandler.obtainMessage(status);
            msg.sendToTarget();
            return MSG_CARD_MUTE;
        }else{
            return MSG_WARM_RESET;
        }
    } catch (IOException | IllegalStateException e) {
        Log.d(TAG,"" + "Exception: " + e.getMessage());
        return MSG_CATCH_EXCEPTION;
    }

}

    II) Set Protocol

    Description – you can choose between T1 or TO

    public int setProtocol(){
    try {
        //Choose protocol (here PROTOCOL_T0)
        mSamReader.setprotocol(PROTOCOL_T0);
        isOnceCompleted = true;
        return MSG_PROTOCOL_SETTING_UP;
    }catch (IOException e) {
        Log.e(TAG, "setProtocol", e);
        return MSG_CATCH_EXCEPTION;
    }
}

    II) Transmit APDU

    Description – Once the above operations are completed, the card is in activated state. Reading and writing operations can then be performed using this function.

    public int transmitAPDU (String apdu) {
    if(apdu.isEmpty()){
        return MSG_EMPTY_EDIT_TEXT;
    } else {
        try {
            byte []  command = Utils.toByteArray(apdu);
            /** Send and receive apdu on SAM reader  **/
            response = mSamReader.ApduTransmit(command);
            return MSG_SEND_RECEIVE_APDU;
        } catch (IOException e1) {
            Log.e(TAG,e1.toString());
            return MSG_CATCH_EXCEPTION;
        }catch (NullPointerException e2){
            Log.e(TAG,e2.toString());
            return MSG_CATCH_EXCEPTION;
        }

    }

}
