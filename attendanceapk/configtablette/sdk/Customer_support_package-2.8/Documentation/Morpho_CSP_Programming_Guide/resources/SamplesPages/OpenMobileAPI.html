I) Initialization

Description – To establish an Open Mobile API communication, you will have to follow those steps :

    1 - Implement a SEServiceCallback

Readers names are :
- SAM
- SMARTCARD
- SIM1
- SIM2

Example - The following code will be used for the SAM reader 

/** Used to connect to OMA service. */
public class SEServiceCallback implements SEService.CallBack {
    @Override
    public void serviceConnected(SEService seService) {
        //List All the readers
        Reader[] readers = seService.getReaders();
        for (Reader reader : readers) {
            if (reader.getName().equals("SAM")) {
                //Initialization of SAM Reader if card is present
                if (reader.isSecureElementPresent()) {
                    samReader = reader;
                }
            } 
        }
    }
}
    2 - Instanciate the SEServiceCallback to initialize the reader.
/** Init to OMA service. */
public void initOpenMobileAPI(View view) {
    updateConsole("*******  Initialisation Open Mobile API  *******");
    if(!transmitTaskPool.isShutdown()) {
        transmitTaskPool.shutdown();
    }
    transmitTaskPool = Executors.newFixedThreadPool(10);
    currentReader = null;
    SEServiceCallback callback = new SEServiceCallback();
    try {
        new SEService(getApplicationContext(), callback);
    } catch (SecurityException e) {
        Log.d(TAG,"Binding not allowed, uses-permission org.simalliance.openmobileapi.SMARTCARD?");
    } catch (Exception e) {
        Log.d(TAG,"Exception: " + e.getMessage());
    }
}

II) Direct Transmit

Description – Only available on SAM and ACS reader, use the following method on a Reader Object to perform the transmission.


Parameters:
byte[] apdu - apdu to be sent to the card.

return value:
byte[] response - response from the card.

response = samReader.ApduTransmit(apdu);

Example - Transmit an APDU from EditText

/** Transmit an APDU directly to the reader
 * Only available on ACS And SAM Readers */
private String directTransmit(String apdu){
    String message = "";
    if (apdu.isEmpty()) {
        message += ("No APDU to send");
    }else {
        try {
            byte[] res = currentReader.ApduTransmit(Utils.toByteArray(apdu));
            if (res == null) {
                restartUi();
                //Power of Sam/Smart card reader
                currentReader.power_off_reader();
                currentReader.hardWare_Reset();
            } else {
                message += ("Send to " + currentReader.getName() + " : " + apdu);
                message += ("\nReceive from " + currentReader.getName() + " : " + Utils.toHexString(res));
            }
        }catch(IOException e){
            message += ("\nCan't transmit apduField");
        }
    }
    return message;
}

III) OMA Transmit

Description – More information can be found in OpenMobileApi Documentation.

    1 - Open a session on a reader object 
session = currentReader.openSession();
    2 - Open a channel with the created session using AID
channel = session.openLogicalChannel(AID);
    3 - Transmit a command with the created channel 
channel.transmit(CMD);

Example - Communication with Open Mobile API.
Note – The AID should match the smartcard applet. 

private String transmitToSim(){
    String message = "";
    try {
        //Opening session for the card
        mSession = currentReader.openSession();
    } catch (IllegalStateException|IOException e) {
        message +=(""+e.getMessage());
    }
    //Getting the ATR of the card (Reset Card)
    byte[] ATR = mSession.getATR();
    message +=("ATR : "+Utils.toHexString(ATR));
    try {
        // Choose the applet to communicate with
        // The AID should be modified to match the applet in the card */
        message +=("\nChoose an AID corresponding to the applet of the card");
        message +=("\nlogical mChannel with AID :"+Utils.toHexString(AID));
        mChannel = mSession.openLogicalChannel(AID);
    } catch (Exception e) {
        message +=("\n"+e.getMessage());

    }
    if (mChannel != null) {
        //Send a command to the applet
        //This need to match the applet command
        message +=("\nChoose The CMD of the applet");
        message +=("\ntransmitting CMD : "+Utils.toHexString(CMD));
        try {
            mChannel.transmit(CMD);
        } catch (IOException e) {
            message +=("\n"+e.getMessage());
        }
    }
    mSession.closeChannels();
    mSession.close();
    return message;
}
