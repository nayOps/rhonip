Functions

The sample provides basic operations for reading and writing to the external storage. Developers shall use this sample to read or write text files stored on an external storage. Sample uses android library. In order to use any network "get/scan" feature developers have to add the three following permissions in the manifest file:

<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE">
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE">
<uses-permission android:name="android.permission.ACCESS_ALL_EXTERNAL_STORAGE">

USB pen drive must be mounted at /storage/uicc0 location to use TOP USB host.
USB pen drive must be mounted at /storage/uicc1 location to use USB host in the docking station.
USB pen drive must be mounted at /storage/usbotg location to use OTG USB host.
/storage/sdcard0 is the SD card mountpoint.
/storage/sdcard1 is the internal SD card mountpoint.

I) Initialization

Description – Instantiate the following strings that are used to access to external storage 

private static final String USB_TOP = 				"/storage/uicc0";
private static final String USB_CRADLE = 			"/storage/uicc1";
private static final String USB_OTG = 				"/storage/usbotg";
private static final String USB_SDCARD = 			"/storage/sdcard1";
private static final String USB_SDCARD_INTERNAL = 	        "/storage/sdcard0";

II) Check storage availability

Description - This method checks if the mounted storage location is available. Since internal SD card is already mounted, this method is not used to check its availability. 

private boolean isUSBAvailable() {
    File usb = new File(storagePath);
    String state = "";
    if (android.os.Build.VERSION.SDK_INT >= android.os.Build.VERSION_CODES.LOLLIPOP) {
        state = Environment.getExternalStorageState(usb);
    }
    if (storagePath.equals(USB_SDCARD_INTERNAL)) {
        state = Environment.MEDIA_MOUNTED;
    }

    return Environment.MEDIA_MOUNTED.equals(state);
}

III) Create file

Description - Create a file on the mounted storage location and write content to it. 

private boolean createFileAndWrite(String boyd){
    File createdFile=new File(storagePath,STORAGE_FILE_NAME);
    try{

        FileWriter writer = new FileWriter(createdFile);
        boyd=boyd.trim();
        writer.write(boyd);
        writer.flush();
        writer.close();

        return true;

    }catch(IOException e){
        Log.e(this.getClass().toString(), "createFileAndWrite", e);
        return false;
    }

}

IV) Read File from external storage

Description - Return the content of the created file. Return "file not found" if the mounted storage location is not available or if the file does not exist.

private String readFileFormUsb() {
    File createdFile = new File(storagePath, STORAGE_FILE_NAME);
    if (isUSBAvailable()) {
        try {
            StringBuilder fileContent = new StringBuilder("");
            byte[] buffer = new byte[1024];
            FileInputStream fis = new FileInputStream(createdFile);
            while (fis.read(buffer) != -1) {
                fileContent.append(new String(buffer));
            }
            fis.close();
            return fileContent.toString().trim();

        } catch (FileNotFoundException e) {
            showToast("File Not Found...");
            Log.e(TAG, "readFileFormUsb", e);
        } catch (IOException e) {
            Log.e(TAG, "readFileFormUsb", e);
            e.printStackTrace();
            showToast("Storage not mounted or not ready to use.");
        }
        return "";
    } else {
        Log.e(TAG,"Storage not mounted in system.");
        showToast("Storage not mounted in system. Please check storage availability");
        return "";
    }
}
