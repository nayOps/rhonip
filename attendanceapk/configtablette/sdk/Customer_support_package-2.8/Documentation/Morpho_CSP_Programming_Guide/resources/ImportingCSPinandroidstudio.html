Importing CSP in android studio

CSP (customer support package) is separated into various sample modules. Developers can import the project in Android Studio and work on the module they want to. In particular, developers need to process as follow to import projects inside Android Studio environment:

1 - Launch Android Studio

2 - Open the CSP CodesSamples directory
