Purpose

The purpose of this part is to describe specific things to configure to use MobIris SDK. As it does acces to system device, it does need specific rights.

Gradle config

To build signed app, it is necessary to configure keystore with system signing key.

build.gradle:

    signingConfigs {
        release {
            storeFile file("../security/keys/platform.jks")
            storePassword "password"
            keyAlias "android"
            keyPassword "password"
        }
    }
    buildTypes {
        release {
            minifyEnabled false
            proguardFiles getDefaultProguardFile('proguard-android.txt'), 'proguard-rules.pro'
            signingConfig signingConfigs.release
        }
        debug {
            signingConfig signingConfigs.release
        }
    }

In manifest, it is necessary to tell that the app has to be launched with system UID

AndroidManifest.xml

<?xml version="1.0" encoding="utf-8"?>
<manifest xmlns:android="http://schemas.android.com/apk/res/android"
    package="com.morpho.irisSample" android:sharedUserId="android.uid.system">

    <uses-feature android:name="android.hardware.camera" />
    <uses-permission android:name="android.permission.CAMERA" />
    <uses-permission android:name="android.hardware.camera.autofocus" />
    <uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
    <uses-permission android:name="android.permission.INTERNET" />
    <!--uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" /-->
...
</manifest>

Network

LKMS license generation does need network access. If the phone doesn’t have acces to wifi network, it is possible to redirect TCP stream by usb port forwarding.

Set local port in LKMS param for IrisInitialize function:

    char providerURL[]="http://localhost:29080/lkms/LicenseRequest?profileId=demoSecureIdentity";
    char lkmsURL[]="http://localhost:29081/lkms-server-app/v1/licenses";
    char licenseFilePath[]="/sdcard/IrisSample/LKMSLicence.dat";

Set port forwarding with adb:

@rem start lkms provider on 29080 port
pushd C:\LKMS\provider\lkms-provider-setup-1.0.2\bin
set CATALINA_HOME=C:\Program Files (x86)\Apache Software Foundation\Tomcat 8.5\
start lkmsadmin.bat start
popd
@rem port forwarding
pushd C:\Users\current\AppData\Local\Android\sdk\platform-tools
set PORT=29080
adb -d reverse tcp:%PORT% tcp:%PORT%
set PORT=29081
adb -d reverse tcp:%PORT% tcp:%PORT%
@rem redirect local 29081 to 8080 lkms server
ssh -L 29081:lkms-server:8080 root@relay.host.com
popd
