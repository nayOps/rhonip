Overview

MobIris is targeted to allow the integrators of the native libraries provided by Morpho who want to use them with iris treatments

The main features are:

  • FotoNation camera integration
  • LKMS licensing
  • Iris capture and Kind7 extraction
  • Iris matching 1/few
  • Native library.

Prerequisites

Skills required

The integration tasks shall be done by developers with knowledge of:

  • Android Studio.
  • Java for Android.
  • Android operating system.
  • Android NDK.

Resources required

Integration may be performed on PC windows or linux or on Mac.

The tools required are:

  • Android studio 2 or above
  • Android SDK tools: preferred latest version (release 24 or above)
  • Android NDK: preferred version 14b
  • Android device (emulator is not supported)
  • Signing key to build system app
