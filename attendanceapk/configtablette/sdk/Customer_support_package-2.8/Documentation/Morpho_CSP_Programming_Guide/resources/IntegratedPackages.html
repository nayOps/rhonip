Integrated SDKs

SmartCard SDK

Introduction

SmartCard SDK is a comprehensive library to make Smart card based android applications.
For more details on Smart card library, please refer to the ACS SmartCard Functions.

SDK Location

SDKs\ACS-EVK-Android-115-A/.

Fingerprint SDK

Introduction

Compact Biometric Module (CBM) is an optical biometric fingerprint sensor device with enhanced security features. CBM allows you to have some security features like :

  • Enrollment
  • Authentication
  • Identification

MorphoSmart SDK library allows developer to access those features.
For more details, please refer to the MorphoSmart SDK Programmer's Guide.

SDK Location

SDKs\MorphoSmart_SDK_6.19.4.0

MobIris SDK

Introduction

Iris sensor device allows you to perform some security features as :

  • Enrollment
  • Identification
  • Authentication

MobIris SDK library allows developer to access those features.
For more details, please refer to the Mobiris SDK Android documentation.

SDK Location

SDKs\MobIris_SDK_1.2.10
