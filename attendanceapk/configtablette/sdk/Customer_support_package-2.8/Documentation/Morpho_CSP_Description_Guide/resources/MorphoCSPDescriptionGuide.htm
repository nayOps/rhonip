MorphoTablet CSP Description Guide

Purpose


The purpose of this Description Guide is to give an introduction to the MorphoTablet-2 or MorphoTablet-2i possibilities as embedded hardware & software.
This guide will help developers to understand MorphoTablet-2(i) context and show the various features associated to this device.
