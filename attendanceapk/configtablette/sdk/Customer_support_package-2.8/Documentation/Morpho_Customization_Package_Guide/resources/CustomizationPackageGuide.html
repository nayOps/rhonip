MorphoTablet Customization Package Guide

Customization package description

MorphoTablet™ 2(i) is a secure tablet that can be customized by third-party companies. It means that third-party companies can develop their own Android applications running on the tablet. This includes the special case of applications requiring system privilegies.

Description

The solution offered by IDEMIA to support MorphoTablet third-party applications is called customization package. A customization package is a special Android OTA package that contains the third-party applications that will be installed to a dedicated partition of the system named integration partition. This partition has a fixed size of 500MiB. Steps to install third-party applications are the following:

  • Populate a predefined file tree with all the third party applications and/or customized settings to be installed on the tablet
  • Generate the associated OTA update package
  • Update the MorphoTablet™ using this OTA package
