MorphoTablet Customization Package Guide

Import customization package into MorphoTablet™ 2(i)

To install the MorphoTablet customization package, please follow the same process as firmware update.
In case of factory reset, integrator applications are restored to the initial version present in integrator partition (integrator partition is not erased by a factory reset).


Two possibilities are available to import the customization package: either from your system file environment (Windows / Linux) or directly from your SD card.

Recovery mode menu

To start recovery mode, press "volume up (+)" button while the tablet is restarting.
Then use "volume up (+) / down (-)" buttons to select and "power" to validate.

When the recovery menu shows up on MorphoTablet, follow those steps to start the update:

  • 1. Connect MorphoTablet to your host machine (Windows / Linux PC)
  • 2. On MorphoTablet, in recovery menu, select "English" then "apply update from ADB"
  • 3. From host machine, run following ADB command line : 
    $ adb sideload mycustpack_update.zip

    • Android menu

  • 1. Copy customization package in tablet root folder, or SD card root folder. Package filename must end with "update.zip", to be reconnized by Android.
  • 2. From MorphoTablet, scroll down the "Settings" menu and click on 'About Tablet'
  • 3. Select "System updates" menu
  • 4. Insert into the tablet your SD card that contains the ready to flash customization package then validate
