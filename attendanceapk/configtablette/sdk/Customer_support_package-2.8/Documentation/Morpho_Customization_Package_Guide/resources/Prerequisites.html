MorphoTablet Customization Package Guide

Prerequisites


Operating System Linux Ubuntu 14.04 (64 bit)
JRE JRE 7 and higher

Run the following command lines:

sudo apt-get update
sudo apt-get install libc6-i386 libstdc++6:i386 openjdk-7-jre lib32z1 android-tools-adb android-tools-fastboot zip
mkdir ~/.android -p
echo 0x225D > ~/.android/adb_usb.ini

Note : You can also launch InstallNeededPackages script to install packages automatically.
